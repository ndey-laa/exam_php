# 📄 CVGEN — Générateur de CV en ligne

CVGEN est une application web interactive permettant de créer et personnaliser facilement un CV professionnel, puis de le *télécharger au format PDF*. Ce projet a été réalisé dans le cadre d’un examen en JavaScript.

---

## ✨ Fonctionnalités

- 📝 Remplissage dynamique des sections :
  - Données personnelles
  - Formations
  - Expériences professionnelles
  - Compétences
  - Langues
- 🎨 Changement de modèle visuel (choix de templates)
- 📄 Génération d’un aperçu de CV en temps réel
- 📥 Téléchargement du CV au *format PDF* fidèle à l'affichage
- ✅ Compatible avec les navigateurs sans serveur local grâce aux CDNs

---

## 🚀 Technologies utilisées

- HTML5
- CSS3
- JavaScript
- [html2canvas](https://html2canvas.hertzen.com/)
- [jsPDF](https://github.com/parallax/jsPDF)

---

## 🧪 Utilisation

1. Ouvrir le fichier index.html dans un navigateur ( Via Live Server)
2. Remplir les différentes sections du formulaire
3. Visualiser en temps réel le rendu du CV
4. Cliquer sur *“Télécharger le CV”* pour générer un fichier Mon-CV.pdf
